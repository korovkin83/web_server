# 
Для установки необходимо:
1. NodeJS
2. Модуль express 
3. запустить скрипт app.js <путь до папки с html,css,js файлами> <порт сервера>
4. перейти в браузер по вашему адрессу(ifconfig.me), к примеру localhost:port 
5. Наслаждаться!
